## 📌 Sobre este proyecto
 
**MOSTO** es una landing page de demostración para un restaurante costarricense de cocina de origen. Este proyecto es un ejemplo del tipo de páginas web que diseña y desarrolla Alejandro Chacón como parte de su servicio de creación de landing pages premium.
 
El objetivo del sitio es mostrar cómo una marca gastronómica puede comunicar identidad, ambiente y oferta a través de una experiencia visual de alta calidad — desde el primer scroll hasta el último botón.
 
---
 
## ✨ Características técnicas
 
- **Cursor personalizado** animado con dot + ring magnético
- **Animaciones de entrada** con GSAP + SplitType (words reveal en el hero)
- **Scroll suave** con Lenis + ScrollTrigger
- **Menú interactivo** por categorías con tabs animados (Entradas, Platos, Postres, Bebidas)
- **Contadores animados** al hacer scroll
- **Botones magnéticos** con efecto de atracción al cursor
- **Galería con drag-scroll** horizontal
- **Navegación dinámica** con glassmorphism al hacer scroll
- **100% single-file** — un solo `.html`, sin build tools, listo para deploy

## 🛠️ Stack
 
| Tecnología | Uso |
